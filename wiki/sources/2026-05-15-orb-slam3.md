# ORB-SLAM3: 视觉、视觉惯性与多地图 SLAM 精确开源库

> Campos 等人在 TRO 2021 发表，ORB-SLAM 系列的最终版。首个基于 MAP 估计的紧耦合视觉惯性 SLAM
> （IMU 初始化也用 MAP），Atlas 多地图系统支持复杂场景鲁棒运行，比前代精确 2-5 倍。

## 核心贡献

1. **基于 MAP 估计的紧耦合视觉惯性 SLAM**：首个在 IMU 初始化阶段也使用 **最大后验概率（MAP）估计**
   的系统，而非传统的松耦合（先 SfM 再加 IMU）。初始化时同时优化所有参数的 MAP 问题，结果显著更准。

2. **Atlas 多地图系统**：跟踪丢失时自动创建新 Active Map，不丢弃旧数据。当重新进入已知区域时，
   通过新型位置识别方法（改进召回率）无缝合并地图。共同区域（Common Regions）检测实现全自动合并。

3. **全传感器配置覆盖**：支持 MONOCULAR / STEREO / RGBD / IMU_MONOCULAR / IMU_STEREO / IMU_RGBD
   六种模式，且支持针孔和鱼眼（fisheye）相机模型。

4. **所有阶段复用全部历史信息**：在 BA 中包含不同时间窗口的共视关键帧，提供高视差观测 ——
   即使时间跨度很大或来自不同建图会话，显著提升精度。

5. **领先精度**：EuRoC 立体惯性平均 3.6 cm；TUM-VI 手持快速运动仅 9 mm。比最接近的竞争对手
   (VI-DSO, VINS-Mono, ROVIO 等) 精确 2-5 倍。

## 关键数学公式

**视觉惯性 MAP 估计（全状态联合优化）**：

$$\arg\min_{\mathcal{X}} \left(\|\mathbf{r}_0\|^2_{\Sigma_0} + \sum_{i\in\mathcal{K}}\sum_{j\in\mathcal{X}_i} \rho\left(\|\mathbf{r}_{ij}\|^2_{\Sigma_{ij}}\right) + \sum_{k=1}^{K-1} \|\mathbf{r}_{\mathcal{I}_k}\|^2_{\Sigma_{\mathcal{I}_k}}\right)$$

其中 $\mathbf{r}_{ij}$ 为视觉重投影残差，$\mathbf{r}_{\mathcal{I}_k}$ 为 IMU 预积分残差，
$\mathbf{r}_0$ 为先验。$\mathcal{X}$ 包含所有关键帧位姿、速度、IMU 偏置和地图点。

**IMU 预积分残差**（Forster 2017 公式，ORB-SLAM3 直接使用）：

$$\mathbf{r}_{\mathcal{I}_{ij}} = \begin{bmatrix}
\mathbf{r}_{\Delta R_{ij}} \\ \mathbf{r}_{\Delta v_{ij}} \\ \mathbf{r}_{\Delta p_{ij}}
\end{bmatrix} = \begin{bmatrix}
\text{Log}(\Delta\tilde{R}_{ij}^T R_i^T R_j) \\
R_i^T(v_j - v_i - g\Delta t_{ij}) - \Delta\tilde{v}_{ij} \\
R_i^T(p_j - p_i - v_i\Delta t_{ij} - \frac{1}{2}g\Delta t_{ij}^2) - \Delta\tilde{p}_{ij}
\end{bmatrix}$$

**最大-MAP IMU 初始化**：

$$\mathcal{X}^* = \arg\min_{\mathcal{X}} \left(\sum_{k=1}^{K-1}\|\mathbf{r}_{\mathcal{I}_k}\|^2_{\Sigma_{\mathcal{I}_k}} + \|\mathbf{r}_p\|^2_{\Sigma_p}\right)$$

在初始化阶段即用非线性优化估计尺度 $s$、重力方向 $R_{wg}$、加速度偏置 $b_a$ 和速度 $\{v_k\}$，不接受
松耦合的近似解。

## 架构概述

```
传感器输入 (Mono/Stereo/RGB-D + 可选 IMU)
        ↓
┌──────────────────────────────────────┐
│           ATLAS (多地图管理器)        │
│  ┌─────────┐  ┌──────────┐          │
│  │Active Map│  │Non-Active │  ...    │
│  │  (当前)  │  │  Maps     │         │
│  └─────────┘  └──────────┘          │
│         ↓ 合并检测                    │
│    Common Regions → Welding BA        │
└──────────────────────────────────────┘
        ↓
[Tracking] ← 视觉惯性紧耦合跟踪
   ├── 纯视觉模式: PnP + Motion-only BA
   └── VI 模式: IMU 预积分预测 + MAP 优化
        ↓
[LocalMapping] ← 局部视觉惯性 BA
        ↓
[LoopClosing] ← DBoW2 + 多地图合并
        ↓
[Full Inertial BA] ← 全局视觉惯性 BA（独立线程）
```

## 关键技术洞察

- ORB-SLAM3 的本质突破在于把"初始化"也纳入 MAP 估计框架 —— 之前的方法几乎都以某种松耦合方式先估计
  初始值再用紧耦合优化，ORB-SLAM3 证明了全 MAP 初始化的可行性和精度优势。
- Atlas 的"丢失不丢弃"策略使得系统在长时间低视觉信息（遮挡、暗光、快速旋转）场景下鲁棒性远超对手，
  是工程上最实用的创新。
