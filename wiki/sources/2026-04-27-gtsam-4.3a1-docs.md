# GTSAM 4.3a1 官方文档

> 面向 GTSAM 4.3a1 使用查询的官方文档整理入口，重点服务 SLAM、navigation、视觉因子和 Python/C++ API 使用。

## 基本信息

- 来源：GTSAM Docs，https://borglab.github.io/gtsam/
- 范围：官方 sitemap 下 230 个页面，以及文档首页链接的 C++ reference
- 基线：按用户要求以 GTSAM 4.3a1 为准；官方首页同时说明 develop 处于 Pre 4.3 模式，存在 4.3 相关弃用与 API 变化
- 重点：使用方式优先，原理解释服务于 API 选择与工程调试

## 核心观点

1. GTSAM 的核心抽象是 factor graph：用 `NonlinearFactorGraph` 表示约束，用 `Values` 表示变量初值或估计，用 optimizer/iSAM2 求解。
2. 机器人中最常用的变量类型是 `Pose2`、`Pose3`、`Rot3`、`NavState`，它们是 manifold/Lie group 类型，更新通常发生在 tangent space。
3. 常用 SLAM 因子包括 `PriorFactor`、`BetweenFactor`、projection/stereo/smart projection factors；navigation 常用 `ImuFactor`、`CombinedImuFactor`、`GPSFactor`。
4. Python 侧使用方式通常是：构图、插入初值、调用优化器、查询结果、用 `Marginals` 或 `ISAM2` 查询不确定性。
5. 自定义因子在 Python 中可用 `CustomFactor`，但必须特别注意 Jacobian 约定、Fortran order 矩阵和 Python callback/GIL 开销。
