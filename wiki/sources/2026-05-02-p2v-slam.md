# P2V-SLAM 隐式点-体素 LiDAR-IMU SLAM

> 这篇公众号文章介绍 T-ASE 2026 论文 `Implicit Point-to-Voxel LiDAR-IMU SLAM`：P2V-SLAM 用神经网络学习局部体素几何关系，替代点到面/点到线等参数化 LiDAR 观测残差，并把残差和不确定度接入 IESKF。

## 核心观点

1. **P2V-SLAM 的问题切入点是 LiDAR 观测模型**：传统 LOAM/FAST-LIO 路线常依赖点到面、点到线或局部高斯分布，复杂植被、曲面、杂乱结构会让显式几何假设失效。
2. **隐式点-体素观测模型把“拟合几何”换成“学习局部几何关系”**：扫描点、邻域体素位置和体素隐式特征进入 IR-Net，输出隐式残差与不确定度。
3. **P2V-SLAM 不是端到端定位网络**：它保留 IMU 传播、IESKF 迭代更新和体素地图管理，只把 LiDAR measurement model 替换成神经隐式观测。
4. **iVoxMap 是在线运行的关键工程结构**：地图按哈希体素管理，每个体素累积固定数量点后由 VE-Net 提取隐式特征，避免每帧全量重算。
5. **主要代价在神经推理**：文章给出的边界是 3.8GHz 8 线程 CPU 上每 100ms 处理约 1000-2000 个有效观测，IR-Net 推理是主要耗时来源。

## 与其他素材的关联

- 与 [[方法-体素地图]] 的关系：P2V-SLAM 把体素从统计/平面拟合单元扩展为神经隐式特征单元，仍保留哈希体素地图的在线更新优势。
- 与 [[方法-IESKF滤波器]] 的关系：P2V-SLAM 没有放弃滤波后端，而是让神经网络输出可线性化的残差和不确定度，再进入 IESKF 更新。
- 与 [[算法-PIN-SLAM]] 的关系：二者都属于神经隐式 LiDAR SLAM 路线，但 PIN-SLAM 更偏连续 SDF 地图表示，P2V-SLAM 更偏学习式 scan-to-map 观测模型。
