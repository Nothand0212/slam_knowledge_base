# VINS-Mono: 鲁棒通用单目视觉惯性状态估计器

> Qin 等人在 TRO 2018 发表，完整单目 VIO 系统方案：鲁棒初始化 + 滑动窗口紧耦合非线性优化 +
> 回环检测与重定位 + 全局位姿图优化。代码开源，支持 PC 和 iOS，是 VIO 工程化标杆。

## 核心贡献

1. **鲁棒在线初始化**：视觉 SfM（滑动窗口内相对位姿 + 3D 路标）→ 视觉惯性对齐（陀螺偏置、速度、
   重力向量、尺度），通过松耦合完成初始值估计后交给紧耦合优化。

2. **滑动窗口紧耦合非线性优化**：在固定大小滑动窗口内联合优化 IMU 残差（预积分）、视觉重投影残差
   和边缘化先验残差，Ceres Solver 求解。

3. **边缘化策略**：滑动窗口新旧帧切换时，对离开窗口的状态执行 Schur 补边缘化，保留其信息作为先验
   残差送入新窗口，避免信息丢失。

4. **回环检测与重定位**：基于 DBoW2 + BRIEF 描述子检测回环 → PnP 几何验证 → 紧耦合重定位
   （回环帧位姿作为常量在滑动窗口中约束当前帧）→ 4-DOF 全局位姿图优化（忽略滚转/俯仰的不可观维度）。

5. **在线外参与时间偏移标定**：相机-IMU 外参（$T_{bc}$）和 IMU-camera 时间偏移（$t_d$）在优化中
   作为可变量在线估计，消除出厂标定误差。

## 关键数学公式

**滑动窗口全状态优化**：

$$\min_{\mathcal{X}} \left\{ \|\mathbf{r}_p - \mathbf{H}_p \mathcal{X}\|^2 + \sum_{k\in\mathcal{B}} \|\mathbf{r}_{\mathcal{B}}(\hat{\mathbf{z}}_{b_k b_{k+1}}, \mathcal{X})\|^2_{\Sigma_{b_k b_{k+1}}} + \sum_{(l,j)\in\mathcal{C}} \rho\left(\|\mathbf{r}_{\mathcal{C}}(\hat{\mathbf{z}}^{c_j}_l, \mathcal{X})\|^2_{\Sigma^{c_j}_l}\right) \right\}$$

其中 $\mathbf{r}_p$ 为边缘化先验，$\mathbf{r}_{\mathcal{B}}$ 为 IMU 预积分残差，
$\mathbf{r}_{\mathcal{C}}$ 为视觉重投影残差（单位球面上的切线平面投影）。

**IMU 预积分残差**：

$$\mathbf{r}_{\mathcal{B}} = \begin{bmatrix}
\delta\boldsymbol{\alpha}^{b_k}_{b_{k+1}} \\ 
\delta\boldsymbol{\beta}^{b_k}_{b_{k+1}} \\ 
\delta\boldsymbol{\theta}^{b_k}_{b_{k+1}} \\ 
\delta\mathbf{b}_a \\ 
\delta\mathbf{b}_g
\end{bmatrix} = \begin{bmatrix}
\mathbf{R}^{b_k}_w(\mathbf{p}^w_{b_{k+1}} - \mathbf{p}^w_{b_k} - \mathbf{v}^w_{b_k}\Delta t_k + \frac{1}{2}\mathbf{g}^w\Delta t_k^2) - \hat{\boldsymbol{\alpha}}^{b_k}_{b_{k+1}} \\
\mathbf{R}^{b_k}_w(\mathbf{v}^w_{b_{k+1}} - \mathbf{v}^w_{b_k} + \mathbf{g}^w\Delta t_k) - \hat{\boldsymbol{\beta}}^{b_k}_{b_{k+1}} \\
2\left[\mathbf{q}^{b_k}_w \otimes (\mathbf{q}^w_{b_{k+1}})^{-1} \otimes (\hat{\boldsymbol{\gamma}}^{b_k}_{b_{k+1}})^{-1}\right]_{xyz} \\
\mathbf{b}_{ab_{k+1}} - \mathbf{b}_{ab_k} \\
\mathbf{b}_{gb_{k+1}} - \mathbf{b}_{gb_k}
\end{bmatrix}$$

**视觉残差（单位球面正切平面）**：

$$\mathbf{r}_{\mathcal{C}} = \begin{bmatrix} \mathbf{b}_1 & \mathbf{b}_2 \end{bmatrix}^T \left( \frac{\hat{\mathcal{P}}^{c_j}_l}{\|\hat{\mathcal{P}}^{c_j}_l\|} - \bar{\mathcal{P}}^{c_j}_l \right)$$

将重投影误差投影到球面正切平面上（2 DOF），消除深度方向的不确定。

**边缘化（Schur 补）**：

$$\begin{bmatrix} \Lambda_{aa} & \Lambda_{ab} \\ \Lambda_{ba} & \Lambda_{bb} \end{bmatrix} \begin{bmatrix} \delta\mathcal{X}_a \\ \delta\mathcal{X}_b \end{bmatrix} = \begin{bmatrix} \mathbf{g}_a \\ \mathbf{g}_b \end{bmatrix} \Rightarrow \underbrace{(\Lambda_{aa} - \Lambda_{ab}\Lambda_{bb}^{-1}\Lambda_{ba})}_{\text{Schur 补}} \delta\mathcal{X}_a = \mathbf{g}_a - \Lambda_{ab}\Lambda_{bb}^{-1}\mathbf{g}_b$$

被边缘化的变量 $\mathcal{X}_b$ 的信息被压缩为 $\mathcal{X}_a$ 上的一阶先验，
确保不丢弃历史约束。

## 架构概述

```
IMU (200 Hz) + Camera (30 Hz)
        ↓
┌─────────────────────────────┐
│     测量预处理               │
│  IMU: 预积分 (两帧之间)      │
│  Camera: KLT 光流 + 关键帧   │
└─────────────────────────────┘
        ↓
┌─────────────────────────────┐
│      初始化 (在线)           │
│  SfM → 视觉惯性对齐         │
│  陀螺 bias → 速度/重力/尺度  │
└─────────────────────────────┘
        ↓
┌─────────────────────────────┐
│   滑动窗口 VIO (Ceres)       │
│  IMU 残差 + 视觉残差 + 先验  │
│  边缘化旧帧, 窗口大小固定     │
│  外参/时间偏移在线标定       │
└─────────────────────────────┘
        ↓ (关键帧 → 回环检测)
┌─────────────────────────────┐
│     回环闭环                 │
│  DBoW2 + PnP + 紧耦合重定位  │
│  4-DOF 全局位姿图优化        │
└─────────────────────────────┘
```

## 关键技术洞察

- VINS-Mono 的边缘化策略使用 **First-Estimate Jacobian (FEJ)**：被边缘化的变量在后续优化中保持
  初始线性化点不变，避免零空间变化引入的虚假信息，这是滑动窗口滤波器的核心工程技巧。
- 4-DOF 位姿图（只优化 $x,y,z,\text{yaw}$，固定 roll/pitch）利用了 VIO 中滚转/俯仰被重力向量
  完全可观的特性，避免回环优化错误修正重力方向。
- 在线外参和时间偏移标定使得 VINS-Mono 对硬件参数误差不敏感，是实际部署时"开箱即用"体验远好于
  其他 VIO 系统的关键差异点。
