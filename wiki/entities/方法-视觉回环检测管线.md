# 视觉回环检测管线

> ORB-SLAM3 的三阶段回环检测与校正管线：从 DBoW2 词袋候选搜索，经过 Sim3 RANSAC 几何验证，到本质图位姿图优化与地图点融合。

## 定义

ORB-SLAM3 的回环检测是一个多阶段管线，同时处理同一地图内的**回环闭合**（Loop Closing）和跨地图的**地图合并**（Map Merging）。核心思想是：词袋检索给出候选，几何验证确认候选，位姿图优化传播校正。

## 三阶段管线

### 阶段 1：候选检测（DBoW2 倒排索引检索）

`KeyFrameDatabase::DetectNBestCandidates` 通过 DBoW2 词袋模型快速检索与当前关键帧共享词汇的历史关键帧。

**核心步骤**（`raw/codes/ORB_SLAM3/src/KeyFrameDatabase.cc:L604-L730`）：

1. **倒排索引查询**：遍历当前帧 BoW 向量中的每个视觉单词，从该单词的倒排列表中收集共享该单词的历史关键帧，排除与当前帧已有共视关系的帧。
2. **共同单词过滤**：保留共同单词数 $\ge 0.8 \cdot \text{maxCommonWords}$ 的关键帧。
3. **DBoW2 得分计算**：对过滤后的候选帧计算词袋相似得分 $s(v_1, v_2)$。
4. **共视得分累加**：对每个候选帧，将其共视邻居（top-10 covisible KFs）的得分累加为 `accScore`，选取得分最高的邻居作为代表帧。
5. **Top-N 选取**：按累加得分排序，取前 N 个候选（默认 $N=3$），按所在地图分为回环候选和合并候选。

同时，若上一关键帧已存在候选（`mnLoopNumCoincidences > 0`），则通过 `DetectAndReffineSim3FromLastKF` 直接在前一候选的邻域追踪回环（`raw/codes/ORB_SLAM3/src/LoopClosing.cc:L535-L576`），而非重新查询 BoW。

### 阶段 2：几何验证（Sim3 RANSAC + 优化 + 一致性检查）

`DetectCommonRegionsFromBoW` 对 BoW 候选执行多级几何验证（`raw/codes/ORB_SLAM3/src/LoopClosing.cc:L578-L896`）：

**2a. BoW 特征匹配**（L591-L691）：
- 对候选帧及其 top-10 共视帧，用 `ORBmatcher::SearchByBoW`（阈值 0.9）匹配 ORB 特征。
- 合并所有共视帧的匹配点，去重，得到合并后的 3D-2D 对应。
- 阈值：$\ge 20$ 个 BoW 匹配才进入几何验证。

**2b. Sim3 RANSAC**（L697-L710）：
- 使用 `Sim3Solver` 进行 RANSAC 迭代（`raw/codes/ORB_SLAM3/src/Sim3Solver.cc:L1-L489`）。
- 每次随机抽取 3 对匹配点，用 Horn 闭式解法计算 Sim(3)（含尺度 $s$）的相对变换。
- 置信度 0.99，最少内点数 $n_{\text{inliers}} \ge 15$，最大迭代 300 次。
- 内点判定：投影误差 $\le$ 阈值。

**2c. Sim3 优化 + 重投影验证**（L746-L793）：
- 将 RANSAC 得到的 Sim3 作为初值，收集候选帧共视区域的所有地图点。
- 用 `ORBmatcher::SearchByProjection`（搜索半径 8px，阈值 1.5）获取重投影匹配。
- 若匹配数 $\ge 50$，执行 Sim3 非线性优化（`Optimizer::OptimizeSim3`，最多 10 次迭代），固定尺度（双目/RGB-D）或不固定（单目）。
- 优化后内点数 $\ge 20$，再用更严格的投影（5px, 1.0）验证，需 $\ge 80$ 个匹配。

**2d. 共视邻域一致性**（L815-L858）：
- 对当前 KF 的共视邻居（top-10），逐一用校正后的 Sim3 做重投影匹配。
- 至少 3 个邻居 KF 通过验证（`nNumKFs >= 3`），才算回环成立。
- 此机制即 **3 次连续一致性检测**：连续 3 个关键帧都确认同一回环区域才最终触发回环校正。

### 阶段 3：位姿图优化与回环校正（CorrectLoop）

`CorrectLoop` 执行回环校正与本质图优化（`raw/codes/ORB_SLAM3/src/LoopClosing.cc:L969-L1213`）：

**3a. 停止局部建图线程**（L969-L999）：
- 请求 LocalMapping 停止，若正在运行 Global BA 则中止之（避免冲突）。

**3b. Sim3 校正传播**（L1007-L1066）：
- 将回环 Sim3 $T_{cw}^{\text{corrected}}$（从当前帧到世界系的带尺度变换）直接赋给当前 KF。
- 对当前 KF 的所有共视邻居 KF，通过相对位姿传播校正后的位姿：
  $$T_{iw}^{\text{corrected}} = T_{ic} \cdot T_{cw}^{\text{corrected}}$$
  其中 $T_{ic} = T_{iw} \cdot T_{wc}$，即 KF i 在世界系到当前 KF 系的变换。
- 同时保存校正前位姿 `NonCorrectedSim3` 和校正后位姿 `CorrectedSim3`。

**3c. 地图点位置校正**（L1069-L1111）：
- 对每个被校正的 KF，用校正前后的 Sim3 变换地图点的世界坐标：
  $$P_w^{\text{corrected}} = T_{wi}^{\text{corrected}} \cdot T_{iw}^{\text{original}} \cdot P_w^{\text{original}}$$
- 即 $\text{g2oCorrectedSwi.map(g2oSiw.map(P3Dw))}$，其中 $T_{wi}^{\text{corrected}} = (T_{iw}^{\text{corrected}})^{-1}$。
- 在 IMU 模式下同时校正速度方向：$v^{\text{corrected}} = R_{\text{cor}} \cdot v^{\text{original}}$。

**3d. 重复地图点融合**（L1116-L1133）：
- 将回环匹配的地图点对（`mvpLoopMatchedMPs` vs 当前帧的地图点）做 `Replace()` 融合。
- 调用 `SearchAndFuse` 在邻域内进一步融合重复点。

**3e. 本质图优化**（L1176-L1184）：
- 检测回环断点之间新出现的共视连接 `LoopConnections`。
- 调用 `Optimizer::OptimizeEssentialGraph` 对本质图进行位姿图优化：
  - 顶点：所有关键帧的 SE(3) 位姿。
  - 边：共视图边 + 生成树边 + 回环边。
  - 优化变量：仅优化位姿（地图点固定），最小化相对位姿误差。
- IMU 模式下使用自由度约束版本 `OptimizeEssentialGraph4DoF`（固定 roll/pitch，仅优化 yaw + xyz + scale）。

**3f. 全局 BA 启动**（L1198-L1207）：
- 若地图较小（< 200 KFs）且在单地图模式下，在新线程中启动全局 BA 进一步精化。

## Agent 实现提示

### 适用场景

- 需要为单目/双目/RGB-D/IMU SLAM 系统实现基于词袋模型的视觉回环检测。
- 需要同时支持回环闭合（同地图）和地图合并（跨地图）两种地点识别模式。
- DBoW2 词袋可作为重定位、回环检测和地图合并的统一候选搜索后端。

### 输入输出契约

| 项目 | 说明 |
|------|------|
| 输入 | 当前关键帧 `mpCurrentKF`（含 BoW 向量、ORB 描述子、地图点关联） |
| 输出 | 若回环检测成功：校正后的关键帧位姿、融合后的地图点、本质图优化结果 |
| 阶段 1 输出 | 回环候选 KF 列表 `vpLoopBowCand` 和合并候选 `vpMergeBowCand` |
| 阶段 2 输出 | 匹配的关键帧 `mpLoopMatchedKF`、Sim3 变换 `mg2oLoopScw`、匹配的地图点对 |
| 阶段 3 输出 | 本质图优化后的全部位姿、融合后的统一地图点集合 |

```
// === 阶段 1：候选检测 ===
function DetectLoopCandidates(KF_current):
    // Step 1a: 倒排索引收集共享单词的 KF
    lKFsSharingWords = []
    for word_id in KF_current.BoW:
        for KF in invertedFile[word_id]:
            if KF not in connectedKFs(KF_current):
                KF.wordCount++ > maxCommonWords and lKFsSharingWords.append(KF)
    
    maxCommonWords = max(KF.wordCount for KF in lKFsSharingWords)
    filter KFs where wordCount >= 0.8 * maxCommonWords
    
    // Step 1b: DBoW2 得分 + 共视累加
    candidates = []
    for KF in filtered:
        score = vocabulary.score(KF_current.BoW, KF.BoW)
        if score >= minScore:
            accScore = score + sum(neighbor.scores for neighbor in KF.covisibles(10))
            candidates.append((accScore, bestKF))
    
    sort candidates by accScore descending
    return topN(candidates, N=3)  // per map

// === 阶段 2：几何验证 ===
function DetectCommonRegionsFromBoW(vpBowCand):
    for pKFi in vpBowCand:
        // Step 2a: BoW 描述子匹配（含共视帧聚合）
        numBoWMatches = matchBoW(currentKF, pKFi + pKFi.covisibles(10))
        if numBoWMatches < 20: continue
        
        // Step 2b: Sim3 RANSAC (min 15 inliers, 0.99 confidence)
        solver = Sim3Solver(currentKF, pKFi, matchedPoints, bFixScale)
        solver.SetRansacParameters(0.99, 15, 300)
        converged = solver.iterate(20)
        if not converged: continue
        
        // Step 2c: 重投影 + Sim3 优化 + 严格验证
        gScw_init = solver.GetSim3()  // R * s | t
        numProj = SearchByProjection(currentKF, gScw_init, mapPoints, 8px, 1.5)
        if numProj < 50: continue
        
        numOpt = OptimizeSim3(currentKF, pKFi, matches, gScw, 10 iters)
        if numOpt < 20: continue
        
        numProj2 = SearchByProjection(currentKF, gScw, mapPoints, 5px, 1.0)
        if numProj2 < 80: continue
        
        // Step 2d: 共视邻域一致性（至少 3 个邻居确认）
        nConsistent = 0
        for neighbor in currentKF.covisibles(10):
            gSjw = g(neighbor, currentKF) * gScw
            if DetectCommonRegionsFromLastKF(neighbor, pKFi, gSjw):
                nConsistent++
        if nConsistent >= 3:
            return true, pKFi, gScw
    
    return false

// === 阶段 3：回环校正 ===
function CorrectLoop(mpCurrentKF, mpLoopMatchedKF, g2oLoopScw):
    // 停止局部建图
    localMapper.RequestStop()
    
    // 传播 Sim3 校正到共视邻居
    CorrectedSim3 = {mpCurrentKF: g2oLoopScw}
    NonCorrectedSim3 = {mpCurrentKF: g2oScw_original}
    
    for KF_i in currentKF.covisibles():
        T_ic = T_iw * inverse(T_cw_original)
        T_iw_corrected = T_ic * T_cw_corrected
        CorrectedSim3[KF_i] = T_iw_corrected
        NonCorrectedSim3[KF_i] = T_iw_original
        KF_i.SetPose(T_iw_corrected)
    
    // 校正地图点
    for KF_i in CorrectedSim3:
        for MP in KF_i.MapPoints:
            Pw_corrected = T_wi_corrected * T_iw_original * Pw_original
    
    // 融合重复地图点
    for (loopMP, curMP) in matchedMPs:
        curMP.Replace(loopMP)
    SearchAndFuse(CorrectedSim3, loopMapPoints)
    
    // 检测新共视连接
    LoopConnections = findNewConnections(currentKFs)
    
    // 本质图优化
    OptimizeEssentialGraph(map, matchedKF, currentKF, NonCorrectedSim3, CorrectedSim3, LoopConnections)
    
    // 添加回环边
    matchedKF.AddLoopEdge(currentKF)
    currentKF.AddLoopEdge(matchedKF)
    
    localMapper.Release()
```

### 实现注意事项

- **minScore 阈值**：DBoW2 的 `minScore` 通常在 0.015-0.03 之间，过高会漏检，过低会产生大量误检候选。
- **候选分组**：将时间上连续的回环候选分组为"一致性组"（`ConsistentGroup`），要求同一组内至少 `mnCovisibilityConsistencyTh=3` 个候选才触发几何验证，防止单帧误检。
- **Sim3 vs SE3**：单目 SLAM 必须使用 Sim(3) 变换（含尺度 $s$），因为单目存在尺度漂移。双目/RGB-D 可固定 $s=1$ 使用 SE(3)。
- **IMU 下的回环验证**：IMU 模式下额外检查 roll/pitch 一致性（$|\phi_x| < 0.008\text{ rad}, |\phi_y| < 0.008\text{ rad}, |\phi_z| < 0.349\text{ rad}$）及尺度范围（$0.9 < s < 1.1$）。
- **同时检测回环与合并**：ORB-SLAM3 的 `NewDetectCommonRegions` 同时运行回环检测和地图合并检测两种模式，合并成功时重置回环候选，回环优先于合并在同一帧被丢弃。
- **地图点校正去重**：通过 `mnCorrectedByKF` 标记避免同一个地图点被重复校正。
