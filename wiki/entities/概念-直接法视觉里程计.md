# 直接法视觉里程计
> Direct Visual Odometry: 用像素光度误差替代特征点重投影误差的视觉里程计范式
## 概述
直接法不使用特征提取和描述子匹配，而是直接比较图像像素的灰度值（光度误差）。代表: DSO (2444★) 使用稀疏光度BA, ROVIO (1245★) 在EKF中使用图像块光度更新, SVO (1568★) 结合直接法前端+重投影后端。
## 核心区别
| | 间接法 (ORB-SLAM3) | 直接法 (DSO) |
|---|---|---|
| 数据关联 | ORB描述子匹配 | 光度一致性 |
| 误差函数 | 重投影几何误差 | 光度误差 = I1(u) - I2(π(T·π⁻¹(u·d))) |
| 优点 | 光照鲁棒, 大基线 | 稠密/半稠密, 无纹理可用 |
| 缺点 | 需纹理 | 光照变化敏感, 需光度标定 |
## 关键变体
- 稀疏直接法: DSO (滑动窗口光度BA)
- 半直接法: SVO (稀疏对齐+特征后端)
- 基于滤波的直接法: ROVIO (EKF+patch光度)
- 延迟边缘化: DM-VIO (DSO+GTSAM, delayed marginalization)
## 对SLAM算法的意义
直接法挑战了SLAM必须依赖特征点的假设。在低纹理、快速运动场景中有独特优势。DM-VIO的延迟边缘化策略证明直接法和因子图可以深度融合。
