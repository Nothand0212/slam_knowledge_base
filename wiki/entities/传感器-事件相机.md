# 事件相机

> 事件相机（Event Camera / DVS / DAVIS）是一种仿生视觉传感器，每个像素独立、异步地检测对数亮度变化，当变化超过阈值时输出事件 (x, y, t, polarity)。

## 核心特性

| 特性 | 事件相机 | 传统帧相机 |
|------|----------|-----------|
| 输出方式 | 异步，像素级独立 | 同步，全局曝光 |
| 时间分辨率 | μs 级 | ms 级（帧率限制） |
| 动态范围 | 120dB+ | 60-70dB |
| 数据带宽 | 仅输出变化（边缘） | 完整帧图像 |
| 运动模糊 | 无 | 曝光时间内有 |

## 事件数据格式

每个事件包含 `(x, y, t, polarity)`：
- x, y：像素坐标
- t：微秒级时间戳（异步，非均匀采样）
- polarity：亮度变化方向（ON=增亮，OFF=变暗）

## 主要挑战

1. 仅输出边缘信息，缺乏纹理区域
2. 噪声多（热噪声、漏电流、背景活动 BA）
3. 数据稀疏，传统帧图像算法不直接适用
4. 需要专门的表示方法（如 Time Surface）

## 在 SLAM 中的应用

- **ESVO**：事件相机立体视觉里程计，使用 Time Surface + ZNCC 块匹配
- Time Surface 将异步事件流转换为类图像的 2D 表示
- 极高时间分辨率和低延迟适用于高速场景（无人机飞行控制等）
