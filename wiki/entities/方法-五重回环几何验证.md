# 五重回环几何验证

> **保留页面**：此页内容已合并到 [[方法-回环验证方法族]]。保留此页以维持现有链接和引用完整性，后续修改请在合并页面进行。

> 4D 雷达回环的多层安全闸：从先验筛选、ISC 描述子、ICP fitness 到单边/多边一致性逐级确认。

## 定义

4DRadarSLAM 针对雷达点云稀疏性设计的严格回环验证流程，通过五层递进过滤确保回环约束的可靠性。

## 核心特征

1. **预筛选**（5 重）：最小回环间距 10m + 累计距离 50m + 气压计高度差 2m + 偏航角差 20° + 椭圆距离模型
2. **ISC 匹配**：Intensity Scan Context 描述子匹配（Ring Key KD 树检索）
3. **ICP Fitness**：配准后 fitness_score < 6（阈值）
4. **Odometry Check**（单边约束一致性）：T_err = T_lc · T_odom，平移 < 0.3/frame，旋转 < 0.05/frame
5. **Pairwise Consistency**（多边约束一致性）：与已有回环边做交叉验证，平移 < 1.5m，旋转 < 0.2rad
- 五层均通过才接受回环，大幅降低误匹配率
- 回环边使用 Huber 鲁棒核（size=1.0），里程计边不使用核
