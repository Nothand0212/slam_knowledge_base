# 曝光在线估计

> FAST-LIVO2 IESKF 状态变量之一：inv_expo_time 作为 19 维状态的第 7 个元素，在每次 VIO 更新中在线估计。

## 设计动机
直接法光度残差对曝光变化敏感。相机自动曝光会导致同一场景的像素亮度变化，传统方法需要假设亮度不变性。FAST-LIVO2 通过在状态中显式建模曝光时间倒数来适应这些变化。

## 数学形式
- 光度残差：`r = τ_cur·I_cur − τ_ref·I_warped_ref`
- τ 的传播模型：常值模型（F_x 中对应列为零）
- 每次 VIO 更新中一起估计 τ 的增量和协方差

## 与传统曝光补偿的区别

传统直接法常使用仿射光度模型或预先曝光标定，把亮度变化作为图像预处理问题。FAST-LIVO2 把曝光时间倒数放进 IESKF 状态，使曝光参数和位姿、LiDAR/视觉残差一起在线更新。

## 与图像预处理的关系

曝光在线估计说明“光度预处理”不一定发生在图像进入前端之前。对于直接法和 LIVO 系统，曝光可以是：

- 离线或在线标定参数，例如 DSO/DM-VIO 的 response、vignette、exposure。
- 残差中的仿射亮度项，例如 SVO/ROVIO 的 gain/offset。
- 优化状态的一部分，例如 FAST-LIVO2 的 `inv_expo_time`。
- 可微渲染中的 per-frame 参数，例如 MonoGS 的 `exposure_a/b`。

这类处理与 CLAHE/均衡化不同：它保留光度残差的可解释性，而不是任意改变图像局部对比度。

## 工程边界

- 曝光在线估计能缓解自动曝光，但不能解决运动模糊、滚动快门或强非线性响应曲线。
- 曝光状态与光度残差强耦合，低纹理区域可能不可观。
- 若相机驱动能提供准确曝光时间，应优先用真实元数据作为先验或初值。

## Agent 实现提示

### 适用场景

当相机自动曝光会破坏直接法亮度一致性，并且系统使用 patch 光度残差、半直接法对齐或 RGB map 颜色一致性时，可以把曝光作为在线估计量、仿射亮度项或带先验的 per-frame 参数。若只有特征点重投影残差，曝光通常不应进入主状态。

### 输入输出契约

- 输入：当前帧图像、参考 patch/参考曝光、当前状态 `inv_expo_time` 或仿射亮度参数、是否启用曝光估计的配置开关。
- 输出：扩展后的残差向量、曝光相关 Jacobian、更新后的曝光状态和协方差、曝光收敛/跳变诊断。
- 状态约定：FAST-LIVO2 使用 inverse exposure；若改成 exposure time 或 affine `a,b`，残差公式和 Jacobian 符号必须同步重写。

### 实现骨架（伪代码）

```pseudo
function add_exposure_to_photometric_update(measurements, state, config):
    for m in measurements:
        I_cur = bilinear_sample(m.curr_image, m.projected_pixel)
        I_ref = m.ref_patch_value

        if config.model == "inverse_exposure":
            r = state.inv_expo_time * I_cur - m.ref_inv_expo_time * I_ref
            J_exposure = I_cur
        elif config.model == "affine_brightness":
            r = exp(state.a) * I_cur + state.b - I_ref
            J_exposure = [exp(state.a) * I_cur, 1]

        append_residual(r)
        append_pose_jacobian(compute_pose_jacobian(m, state))
        if config.estimate_exposure:
            append_exposure_jacobian(J_exposure)

    state = eskf_or_optimizer_update(state, residuals, jacobians, priors)
    return state, exposure_diagnostics
```

### 实现注意事项

- 曝光状态应有合理初值和先验；没有纹理或大量饱和像素时，曝光与位姿可能共同退化。
- 使用 inverse exposure 时，`r = tau_cur * I_cur - tau_ref * I_ref` 的 `J_tau = I_cur`；如果写成 `I_cur / exposure`，符号和尺度会变化。
- 若相机元数据提供真实 exposure time，可作为观测或先验，而不是完全依赖视觉残差估计。
